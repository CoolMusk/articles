ECMAScript中有6种简单数据类型：

- Undefined
- Null
- Boolean
- Number
- String
-  Symbol (更新： ECMAScript 6 新定义类型)

1种复杂数据类型：

- Object

ECMAScript不支持任何创建自定义类型的机制，而所有值最终都将是上述6种数据类型之一。
